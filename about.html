About

Senior Software Engineer at Microsoft (Fabric Spark) — LLM agents, RAG, and diagnostics for big-data systems.

Download résumé (PDF) Email me

I'm Ethan — a senior software engineer with about ten years of experience building large-scale data platforms, and more recently the AI systems that make them smarter. I'm based in Palo Alto, California.

At Microsoft I work on the Fabric Spark team, where I built an LLM-powered Spark Diagnostic Agent from zero to one: in-line code optimization, automated error-fix suggestions, and performance tuning on a platform that runs more than 10M Spark applications a day. The work spans agent orchestration, RAG, evaluation harnesses, and real-time diagnostics, with results published at the VLDB 2025 AIDB Workshop.

Before Microsoft I spent years deep in big data — real-time Flink analytics frameworks, customer data platforms, and billion-event ETL pipelines at JD.com and Xiaomi.

Highlights

  • Built a 0→1 LLM Spark Diagnostic Agent on Microsoft Fabric Spark — a platform running 10M+ Spark applications a day.
  • Published at the VLDB 2025 AIDB Workshop (Spark Run Series Analysis).
  • Open-source contributor to microsoft/skills-for-fabric.

Experience

  • Microsoft — Senior Software Engineer (2022–present). AI-powered diagnostics and developer tooling for Fabric Spark: a 0→1 Spark Diagnostic Agent, a 300+ case evaluation harness mined from production incidents, a RAG knowledge system grounded by vector search, Spark Run Series Analysis (VLDB 2025 AIDB Workshop), and Spark Advisor — a real-time engine surfacing optimization hints in notebook output.   • Unicorns.co — Software Engineer (2016–2018). Built high-throughput web crawlers collecting billions of e-commerce records daily across anti-crawler defenses — encryption, IP limits, CAPTCHA — and designed an anti-crawler defense system to protect proprietary data assets.
